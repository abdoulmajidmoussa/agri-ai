


# 🌱 AGRI-IA — Conseiller Agricole Intelligent

> Assistant IA pour les petits exploitants agricoles d'Afrique de l'Ouest

## 🎯 Problème résolu
Les petits agriculteurs des zones rurales n'ont pas accès à des 
conseils agricoles professionnels en temps opportun. Les maladies, 
ravageurs et aléas climatiques compromettent leurs récoltes sans 
qu'ils puissent réagir à temps.

## 💡 Solution
AGRI-IA est un assistant vocal et textuel qui permet à tout 
agriculteur — même non scolarisé — de :
- Diagnostiquer les maladies de ses cultures via photo ou texte
- Recevoir des conseils adaptés au contexte local
- Consulter les réponses hors ligne
- Interagir en français, wolof, bambara, mooré ou hausa

## 🛠️ Technologies utilisées
- Google AI Studio
- Gemini 3.1 Pro Preview
- React
- Gemini API

## 📱 Fonctionnalités
- Chat multilingue (FR / Wolof / Bambara / Mooré / Hausa)
- Analyse de photos de cultures
- Diagnostic des maladies et ravageurs
- Conseils avec produits locaux disponibles
- Historique consultable hors ligne
- Interface adaptée aux non scolarisés

## ⚙️ Installation locale
1. Clone ce repo : `git clone https://github.com/TON_USERNAME/agri-ia`
2. Installe les dépendances : `npm install`
3. Crée un fichier `.env` et ajoute : `GEMINI_API_KEY=ta_clé_ici`
4. Lance : `npm start`
